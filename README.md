# LEPS_CASINO 🎰

Телеграм-бот казино с игрой "Мины", реферальной системой и админ-панелью.

## 🎮 Функции

- **Игра "Мины"** - выберите 2-15 мин, открывайте безопасные ячейки
- **Реферальная система** - получайте 10% от проигрышей приглашенных друзей
- **Админ-панель** - управление пользователями и балансами
- **Статистика** - отслеживайте вашу игру
- **CryptoBot интеграция** - пополнение баланса криптовалютой

## 🚀 Быстрый старт на Heroku

### 1️⃣ Подготовка на Heroku

1. Откройте https://www.heroku.com и залогиньтесь
2. Нажмите **+ New** → **Create new app**
3. Название: `leps-casino-bot`
4. Выберите регион (Europe)
5. Нажмите **Create app**

### 2️⃣ Подключение GitHub

1. На странице приложения перейдите в **Deploy**
2. Выберите **GitHub** как способ развертывания
3. Нажмите **Connect to GitHub**
4. Найдите репозиторий `LEPS_CASINO`
5. Нажмите **Connect**

### 3️⃣ Добавление переменной окружения

1. Перейдите в **Settings**
2. Нажмите **Reveal Config Vars**
3. Добавьте:
   - **Key**: `TELEGRAM_BOT_TOKEN`
   - **Value**: `ВАШ_НОВЫЙ_ТОКЕН_БОТА`
4. Нажмите **Add**

### 4️⃣ Развертывание

1. Вернитесь в **Deploy**
2. Прокрутите вниз до **Manual deploy**
3. Выберите ветку `main`
4. Нажмите **Deploy Branch**
5. Дождитесь завершения ("Your app was successfully deployed")

### 5️⃣ Активация

1. Нажмите **More** (верхний правый угол)
2. Выберите **View logs** (проверьте что нет ошибок)
3. Нажмите **More** → **Restart dynos**
4. Выберите тип **Worker**
5. Включите переключатель ✅

## 🎯 Использование

- `/start` - начать игру
- `/admin` - админ-панель (только для администратора)
